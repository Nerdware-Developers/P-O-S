## üîß Troubleshooting

### "Add to Home Screen" option not showing?

**For Android:**
- Make sure you're using **Chrome** browser
- Try refreshing the page
- Clear browser cache and try again
- Make sure the website has loaded completely

**For iPhone:**
- **MUST use Safari** - Chrome won't work for this
- Make sure you're on the actual website (not a preview)
- Try refreshing the page
- Make sure the website has loaded completely

### App icon not appearing?

- Check your home screen - it might be on a different page
- Look in your app drawer (Android) or search for "POS"
- Try installing again

### App opens in browser instead of fullscreen?

- Make sure you opened it from the **home screen icon**, not from browser bookmarks
- Delete the shortcut and reinstall it
- On Android, go to Chrome settings ‚Üí "Add to Home screen" and make sure it's enabled
